# 语料技法库(corpus-craft)

从标杆作品逐本检索取证后提炼的可复用技法,供起草(novel-write)与评审(novel-review)引用。语料定位见 design doc 的语料表。

**总纲(keystone 重新框定)**:金庸 + 基督山伯爵都在"报纸日更连载约束下两端通吃"(留存 + 文学深度),消解了"病毒传播 vs 文学性"的二元对立。rubric 北极星 = 以它俩为默认锚点,不在两极间二选一。

## 加分技法

### 复仇/恩仇结构模板(基督山伯爵)
- **比例**:1/4 铺垫被陷害 → 3/4 复仇展开。
- **多线**:多条复仇线交叉,保持独立,后期才汇合(交叉而不乱)。
- **手段**:靠谋略让仇人作茧自缚,**克制巧合**(不靠天降好运)。
- **双线**:报恩与复仇并行,构成"善恶终有报"的完整伦理闭环。
- 适用:复仇流网文(仙逆即复仇驱动)。

### 伏笔工具箱:草蛇灰线五法(红楼梦)
1. **谐音法**:人名/地名藏义(甄士隐=真事隐,贾雨村=假语村言)。
2. **谶语法**:事后应验的话/判词。第五回判词 = 群像命运总纲。
3. **影射法**:角色互为影子(晴为黛影,袭为钗副)。
4. **引文法**:前文某情节像引子,引导后文另一情节。
5. **化用典故法**。
- 配合 bible 伏笔账本:每条伏笔标埋设章 + 计划回收章 + 状态。

### 悬念靠视角制造(呼啸山庄)
- **三重嵌套叙述**:外来者(Lockwood)有限视角埋疑 → 知情者(Nelly)倒叙解谜。
- 外来者的好奇心 = 读者的好奇心,层层悬念牵引追读。
- 同一事件多视角"肯定→否定→再否定"逐层剥茧(亦见金庸暴风雪山庄模式)。

### 连载留存机制(金庸)
- 每日必须留钩:每章末"欲知后事"式悬念吊胃口。
- 人物用反转塑造(岳不群)+ 群像对比(好人各有不同,坏人各有特色)。
- 三角情缘把个人关系架在民族/国家存亡边缘,放大张力。
- 景物烘托情绪(未着一字写悲,景物写尽悲)。

### 情绪循环单元(完美世界)
- 可复制单元:**冲突铺垫 → 装逼打脸高潮 → 路人甲旁白放大爽感**。
- 地域递进升级:每到新图引入更强对手,推动境界跃升。

### 护城河:套路里注入情感与哲思(仙逆)
- 纯套路不封神。爽文骨架 + 情感羁绊(爱情/亲情/友情/师徒)+ 哲思名场面("化神先化凡")。
- 章末钩子标题画面化 + 链式危机(闯关→险死→引出更大反派)。

### 场景级意境(雪国,最窄用法)
- 物哀 / 留白 / 余情:名场面的氛围与意象(雪、镜、火)。
- **严格限制**:仅用于场景级名场面文笔,**严禁作为通篇驱动**——纯留白无情节对网文致命。

## 反面警示(rubric 扣分项)

| 警示 | 出处 | 判定 |
|------|------|------|
| 前期节奏过慢劝退 | 剑来、仙逆前期 | 黄金三章/早期章节铺垫过载、主线推进慢 |
| 主角光环过强、张力崩 | 完美世界 | 同境无敌、毫无失败风险 |
| 人设前后崩坏 | 仙逆后宫毁"守男德" | 角色行为与已立人设矛盾 |
| 审美疲劳、无新鲜感增量 | 烽火自述 | 长期维持同一水准 → 读者自动降分;rubric 要测"增量"非绝对质量 |
| 滥用巧合 | 基督山反面 | 靠天降好运而非谋略推进 |
| 纯留白无情节 | 雪国式误用 | 通篇意境无爽点无钩子,网文致命 |
