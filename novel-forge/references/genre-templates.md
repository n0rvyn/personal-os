# 流派模板库(genre-templates)

按流派给爽点类型库 + 套路模板,供立项(选流派)、分卷章纲(选套路)、起草(命中流派爽点)、评审(按流派基准)引用。novel-deconstruct 拆书的提炼结果也回灌到这里扩充。

**当前 seed:仙侠、玄幻**(crystal P8 Chosen:先 seed 最常见、匹配研究语料的两个;其余流派后续迭代扩充)。

## 仙侠

- **常见母题**:凡人流逆袭、复仇、寻道/问道、宗门成长。
- **爽点类型库**:越境界战胜、打脸宗门弃徒待遇、夺宝/抢机缘、护短杀伐、问道顿悟。
- **金手指范式**:神秘珠子/老爷爷/异瞳/上古传承(必设**限制**,防失控)。
- **升级骨架**:境界阶梯(炼气→筑基→金丹→元婴→…)+ 地域递进(凡间→宗门→上界)。
- **节拍参考**(语料:仙逆):凡人被嘲讽→得金手指→闯关→险死还生→得传承→引出更大反派(链式危机)。情感羁绊(爱情/亲情/师徒)作护城河。
- **反面警示**:前期主角只知修炼日子枯燥易劝退(仙逆教训);后宫处理崩"深情"人设。

## 玄幻

- **常见母题**:草根逆袭、家族/部族崛起、太古秘辛、争霸登顶。
- **爽点类型库**:装逼打脸(冲突→打脸→旁白放大,语料:完美世界)、跨级战、夺造化、震慑天骄、格局揭秘(世界观恢弘带来的"格局爽")。
- **金手指范式**:特殊体质/血脉/重瞳/吞噬类(设限制)。
- **升级骨架**:境界跃升 + 空间递进(下界→上界→仙域),每到新图引入更强对手。
- **节拍参考**(语料:完美世界):苦情身世建代入→草根逆袭统一下界→跨界升级引更强敌→装逼打脸高潮→守护诸天升华。
- **反面警示**:主角同境无敌→张力不足(完美世界教训);配角脸谱化。

## 扩充约定
- 本文件是 plugin 内的**静态 seed 参考**(运行时只读)。新流派由 **plugin 维护时手动追加**——可参考 `novel-deconstruct` 拆爆款的提炼结果(该结果运行时落 cwd,不自动回灌进这里;cwd 模型 + plugin 缓存只读)。
- 每个流派至少含:母题 / 爽点类型库 / 金手指范式 / 升级骨架 / 节拍参考 / 反面警示 六项。
