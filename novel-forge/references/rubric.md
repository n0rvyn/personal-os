# Rubric:网文章节评审标准(crystal D-004 / D-005 / D-006)

rubric 既是**评审标准**,也是**起草目标**(写的目标 = 评的标准,闭环)。起草(novel-write)和评审(novel-review 四镜头)读同一份。

## 评分形态(D-006)

- 每轴定性三级:**达标 / 偏弱 / 缺失**,**必须附证据**(引用本章具体段落)+ 一句改进建议。
- **不打数字分**——散文打 0-100 是假精度、不可行动。
- **留存轴只给预测代理,不给真实指标**:真实追读率/完读率上线后由读者行为测出,rubric 给不了。它产出的是预测,如"黄金三章钩子:偏弱""预计流失点:第 2 章中段铺垫过长",且**必须标注是预测**。
- 整合:高权重轴上"偏弱/缺失" → must-fix;**连续性矛盾(硬伤)恒 must-fix,不论权重**(见 novel-review)。

## 三组分层轴(D-004)

### 留存轴(病毒传播极,网文主导)
权重随定位"爽↔文滑块"变化。
| 轴 | 判定要点 |
|----|---------|
| 开局钩子/黄金三章 | 聚焦主角、强冲突拉主线、信息量克制(不一股脑塞设定)、先抑后扬建代入。仅第 1-3 章 |
| 爽点节拍 | "情绪循环单元"频率与强度(冲突铺垫→打脸/兑现→放大);对照 bible 爽点节拍账,距上次爽点章距是否过长(爽点荒) |
| 章末钩子 | 是否留住读者读下一章;钩子是否画面化/悬念化;对照 bible 避免钩子类型重复 |
| 升级开图 | 境界/地域是否有推进感;新图是否引入更强对手 |

### 结构桥轴(两端通吃,**恒定高权重、不随定位变** D-004)
| 轴 | 判定要点 |
|----|---------|
| 伏笔草蛇灰线 | 对账 bible 伏笔账本:本章埋的伏笔是否入账、回收的是否与埋设一致、是否有该收未收(参 corpus-craft 红楼五法) |
| 多线群像 | 多线是否交叉而不乱、配角是否脸谱化 |
| 复仇/恩仇结构 | 复仇/恩怨推进是否靠谋略而非巧合、是否作茧自缚式(参 corpus-craft 基督山模板) |
| 叙述视角 | 视角是否稳定、是否用视角制造悬念(参 corpus-craft 呼啸嵌套) |

### 深度护城河轴(文学性极,可配置;权重随滑块变)
| 轴 | 判定要点 |
|----|---------|
| 人物烈度/反转 | 人物是否有烈度/复杂性/反转,而非工具人 |
| 情感真实/羁绊 | 情感是否落地、羁绊是否能引共鸣(护城河:纯套路不封神) |
| 主题统一 | 本章是否服务全书主题、有无升华空间 |
| 场景文笔意境 | **最窄轴、最低权重**。名场面的文笔/意境(参 corpus-craft 雪国,仅场景级,严禁通篇留白驱动) |

## 定位 → 权重导出表(D-005)

结构桥轴**恒高**(权重 3,所有档不变)。滑块只调"留存 vs 护城河"的相对权重:

| 爽文档位 | 留存轴 | 结构桥轴 | 深度护城河轴 |
|---------|--------|---------|-------------|
| 纯爽 | 3(最高) | 3(恒高) | 1(最低) |
| 偏爽 | 3 | 3 | 2 |
| **均衡(默认 = 金庸·基督山锚点)** | **2-3** | **3** | **2-3** |
| 偏文 | 2 | 3 | 3 |
| 纯文 | 1 | 3 | 3(最高) |

- 权重含义:3 = 该组轴上"偏弱/缺失"必进 must-fix;2 = 进 must-fix 需明显;1 = 一般归 nice-to-have。
- **均衡档是默认锚点**:金庸/基督山证明日更连载约束下留存与文学深度可同时拿到,不是二选一。
- 流派维度选爽点类型库 + 套路模板(见 corpus-craft);平台维度设留存基准线(日更字数、追读阈值),不改三组相对权重。

## 反面警示(扣分项,见 corpus-craft 详述)
节奏过慢劝退、主角光环过强张力崩、人设崩坏、审美疲劳(无新鲜感增量)、滥用巧合、雪国式纯留白驱动。
